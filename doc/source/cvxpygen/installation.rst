.. _cvxpygen-installation:

Installation
============

To install CVXPYgen, run:

.. code-block:: bash

    pip install cvxpygen

Additional Requirements
-----------------------

If you wish to use the ``Clarabel`` solver, you need to install:

- `Rust <https://www.rust-lang.org/tools/install>`_
- `Eigen <https://github.com/oxfordcontrol/Clarabel.cpp#installation>`_

For Examples
------------

The example notebooks located in `examples/ <https://github.com/cvxgrp/cvxpygen/blob/master/examples/>`_ require ``matplotlib``.

.. note::
    **Windows users:** CVXPYgen is tested with Visual Studio 2019 and 2022. Newer and older versions might work as well.
